
# Afrus Test Front end

Se desarrollo una interfaz utilizando Vue 3, Typescript, GSAP y PrimeVuepara la gestion de Productos y compradores

**Live Demo:** https://afrus-front.vercel.app

**Codigo API:** https://github.com/AlfonsoCasajus/afrus-api

## Tecnologias Utilizadas

**Interfaz:** Vue 3 (Composition API), Typescript, PrimeVue

**Animaciones:** GSAP

**Integracion API:** Axios


## Correr Localmente
Clonar Proyecto

```bash
  git clone https://github.com/AlfonsoCasajus/afrus-front.git
```

Entra al Directorio del Proyecto

```bash
  cd afrus-front
```

Instala las dependencias

```bash
  npm install
```

 Inicia el Servidor

```bash
  npm run dev
```


## Varbiales de Entorno

En el archivo .env.example se econtraran la siguiente variable de entorno que debe ser agregada al archivo .env

`VITE_API_URL=`
